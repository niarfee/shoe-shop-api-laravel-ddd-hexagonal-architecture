## Project Creation 🌱

### Create Laravel Sail skeleton with Docker:

The following command will create the Laravel skeleton with the
`docker-compose.yml` file in the current directory:

```
curl -s "https://laravel.build/shoe-shop-api-laravel-ddd-hexagonal-architecture?with=mysql&php=84" | bash
```

### Install symfony/uid:

```
composer require symfony/uid
```

### Install barryvdh/laravel-ide-helper:

```
composer require --dev barryvdh/laravel-ide-helper
```

### Install friendsofphp/php-cs-fixer:

```
composer require --dev friendsofphp/php-cs-fixer
```

### Install laravel/sanctum:

```
composer require laravel/sanctum
```
